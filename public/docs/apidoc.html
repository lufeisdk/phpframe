内置Jquery，打开F12可以进行操作，比如说 $.ajax() 。

用户登录接口
POST api/login/signon
请求参数 参数类型 参数说明
$username string 用户名
$passwd string 密码
返回名称 返回类型 返回说明
$json json {'code':200,'msg':'登录成功',"data":{"token":"token值"}}
$json json {'code':500,'msg':'登录失败'}
$json json {'code':500,'msg':'用户名或密码不能为空'}
$json json {'code':500,'msg':'用户名不存在'}
$json json {'code':500,'msg':'密码输入错误'}
用户注册接口
POST api/register/signin
请求参数 参数类型 参数说明
$username string 用户名
$nickname string 昵称
$passwd string 密码
返回名称 返回类型 返回说明
$json json {'code':200,'msg':'恭喜你，注册成功'}
$json json {'code':500,'msg':'不好意思，注册失败了'}
$json json {'code':500,'msg':'用户名或密码不能为空'}
$json json {'code':500,'msg':'用户名已经注册，请更换其他用户名'}
用户信息接口
POST api/user/info
请求参数 参数类型 参数说明
$token string 接口token值
返回名称 返回类型 返回说明
$json json {'code':200,'msg':'ok',"data":{"用户数据"}}
$json json {'code':500,'msg':'用户查询失败'}
$json json {'code':500,'msg':'用户信息获取失败'}
更新密码接口
POST api/user/updatePasswd
请求参数 参数类型 参数说明
$passwd string 密码
$passwd_confirm string 确认密码
返回名称 返回类型 返回说明
$json json {'code':200,'msg':'密码更新成功'}
$json json {'code':500,'msg':'密码更新失败'}
$json json {'code':500,'msg':'用户信息查询失败'}
$json json {'code':500,'msg':'两次密码输入不一致'}
$json json {'code':500,'msg':'密码输入不能为空'}
